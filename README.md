
# challange

A brief description of what this project does and who it's for



## Getting Started

To run the app, make sure you have Docker installed. Then, start the application with:


```bash
  docker compose up

```
you need to wait 20 sec until elasticsearch to be up and running


## Monitoring

#### Sidekiq Queue

To monitor the Sidekiq queue, visit: http://localhost:3000/

#### Database Management

To monitor and manage the database, visit: http://localhost:8086/

Note: To connect to db from adminer use in host field localhost if it does not work, use IP:3306.

#### API Testing
